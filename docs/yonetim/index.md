# Yönetim

Yönetim bölümü, iyibir WMS'in sistem yönetimi, kullanıcı yönetimi, barkod şablonları ve bildirim ayarlarını kapsar.

---

## Bölümler

| Bölüm | Açıklama |
|---|---|
| [Kullanıcı Yönetimi](/docs/yonetim/kullanici-yonetimi) | Kullanıcı oluşturma, düzenleme, şifre sıfırlama |
| [Yetki Matrisi](/docs/yonetim/yetki-matrisi) | Rol bazında hangi operasyonlar yapılabilir |
| [Barkod Şablonları](/docs/yonetim/barkod-sablonlari) | Etiket tasarımı ve yazıcı ayarları |
| [Bildirim Ayarları](/docs/yonetim/bildirim-ayarlari) | Push ve e-posta bildirim yapılandırması |

---

## Yönetim Erişim Seviyeleri

| Bölüm | Admin | Yönetici | Operatör |
|---|---|---|---|
| Kullanıcı Yönetimi | Tam | Görüntüle | Hayır |
| Yetki Matrisi | Tam | Görüntüle | Hayır |
| Barkod Şablonları | Tam | Kullan | Kullan |
| Bildirim Ayarları | Tam | Kendi | Kendi |
| Sistem Ayarları | Tam | Hayır | Hayır |

:::info
Bu bölümdeki işlemlerin büyük çoğunluğu web arayüzünden yapılır. Mobil uygulamadan yalnızca kendi bildirim tercihlerinizi düzenleyebilirsiniz.
:::
