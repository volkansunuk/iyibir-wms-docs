# Web Arayüzü (Blazor)

iyibir WMS web arayüzü, **Blazor Server** teknolojisi ile geliştirilmiş, gerçek zamanlı veri güncellemesi sunan bir yönetim ve raporlama platformudur. Tarayıcı üzerinden çalışır; herhangi bir ek kurulum gerektirmez.

---

## Erişim

Web arayüzüne herhangi bir modern tarayıcı ile erişebilirsiniz:

```
https://<sunucu-adresi>/
```

Desteklenen tarayıcılar: Chrome 90+, Edge 90+, Firefox 88+, Safari 14+

## Web vs Mobil Karşılaştırması

| Özellik | Web Arayüzü | Mobil Uygulama |
|---|---|---|
| Operasyon Gerçekleştirme | Kısıtlı | Tam |
| Raporlama | Tam | Kısıtlı |
| Kullanıcı Yönetimi | Tam | Yok |
| Dashboard | Tam | Özet |
| Barkod Tarama | Hayır | Evet |
| Offline Mod | Hayır | Evet |
| Depo Yapılandırma | Tam | Yok |

:::info
Web arayüzü ağırlıklı olarak **yöneticiler ve depo müdürleri** tarafından raporlama, yapılandırma ve denetleme amaçlı kullanılır. Sahada çalışan operatörler için mobil uygulama birincil araçtır.
:::
